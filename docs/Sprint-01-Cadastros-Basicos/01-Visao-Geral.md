# Sprint 01 - Cadastros Básicos

Esta sprint implementa a base cadastral para o futuro módulo de produtos da Orizon Distribuidora.

Cadastros entregues:

- Categorias
- Subcategorias
- Marcas
- Unidades de medida
- Grupos de produtos
- Fornecedores
- Parceiros comerciais
- Depósitos
- Localizações internas
- Status de cadastro

A implementação preserva a arquitetura existente em Domain, Infrastructure e Web, usando ASP.NET Core MVC, EF Core e o layout administrativo Orizon UI.
